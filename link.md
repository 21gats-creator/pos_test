# Links

## Omise / Opn Payments
| | URL |
|---|---|
| Dashboard | https://dashboard.omise.co |
| API Keys (test & live) | https://dashboard.omise.co/settings/keys |
| test key | ขึ้นต้นด้วย `skey_test_` / `pkey_test_` |
| live key | ขึ้นต้นด้วย `skey_live_` / `pkey_live_` |

## Supabase
| | URL |
|---|---|
| Dashboard | https://supabase.com/dashboard |
| API Keys (URL + anon key) | Dashboard → Project → Settings → API |
| Edge Functions | Dashboard → Project → Edge Functions |
| Secrets (OMISE_SECRET_KEY) | Dashboard → Project → Settings → Vault |

## แก้ไขค่าตั้งต้น
ไฟล์ `env.js` — ใส่ค่าของตัวเองทั้งหมด
ไฟล์ `api/config.php` — ใส่ Omise Secret Key (สำหรับ XAMPP)
